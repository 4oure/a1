Chicago's Justice Gap

Crime in Chicago fell nearly 50% since 2001 — but the arrest rate fell even faster. Brush Chart 1 to filter Charts 2 and 3 by year range.

Chart 1 — Crime Volume & Arrest Rate (2001–2025)

Bars = total crimes. Line = arrest rate %. Brush to filter the charts below.

Chart 2 — Crime Type Breakdown (All Years)

Total crimes vs. arrests per crime type. Updates when you brush Chart 1.

Chart 3 — Arrest Rate by Crime Type (All Years)

Which crimes lead to arrests? The most common crimes are the least likely to be solved.

About the Dataset

This visualization uses the Chicago Crimes dataset (2001–present) from the Chicago Data Portal, extracted from the Chicago Police Department's CLEAR system. The raw dataset contains over 8.5 million rows. We pre-aggregated it to yearly counts by crime type for browser performance.
